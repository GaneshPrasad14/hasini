# Hasini Eye Clinic and Optics

A modern web application for Hasini Eye Clinic and Optics — providing expert eye care, premium eyewear, and comprehensive eye examinations since 2010.

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Backend**: Node.js + Express
- **Database**: MongoDB Atlas

## Getting Started

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm start
```

The frontend runs at `http://localhost:8080` and the backend at `http://localhost:7080`.
